# 游戏大概思路

## 设计3个类型角色
### 1.近战 亚瑟(?)
    拥有普通的攻击力，较高生命值和防御力
    攻击力：500-150
    防御力：100
    生命值：10000
### 2.远程 公孙离(?)
    拥有较高攻击力，但生命值和防御力偏低,攻击附带10%吸血
    攻击力：500+150
    防御力：100-30
    生命值：7000
### 3.中距离法师 奕星(?)
    攻击无视防御，生命值和防御力偏低
    攻击力：300
    防御力：100-50
    生命值：650
## 伤害计算方式
    伤害 = 攻击力 * (敌方防御力/(100+敌方防御力))
